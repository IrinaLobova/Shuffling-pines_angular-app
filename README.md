Build: 
1. npm install
2. bower install
3. gulp
Open localhost:8080. 

How to use the app: 

1. fill the form, check the information you entered on the right side of the screen.
   Then click submit button. Enter as many guests as you want.
2. To see the list of all guests you entered switch to the Guests tab.
3. Here you can edit any of your entries. To edit name click on it, make changes and click 
   save. To edit date simply click on it, the data-picker will pop up. You can also edit 
   transit status and location. 
4. You can remove any guest from the list. Click remove, then confirm.
